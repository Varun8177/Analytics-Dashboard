A responsive and interactive dashboard that visualises data from a fictional e-commerce
platform.

# Installation
 1. clone the repository
 2. `npm install`
 3. add env variables
    - `REACT_APP_BASEURL=https://analytics-dashboard-backend-drab.vercel.app`
 4. `npm run start`

## Tech Stack Used

- React.js
- Chart.js
- Tailwind CSS
- axios
- react-icons
